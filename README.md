# Odoo-estate-module

# Requirements

1. Python (preferably version 3.7 or later)
2. PostgreSQL (a relational database)
3. Git (for version control)
4. Node.js and npm (for frontend assets)

# clone the repository locally

```
git clone https://github.com/BakrAsskali/Odoo-estate-module/
```

# Create a virtual python environment

```
python3 -m venv odoo-venv
```

# Start odoo server

```
./odoo-bin -c odoo.conf
```

# Access odoo server in the follwing address

```
http://localhost:8069
```
